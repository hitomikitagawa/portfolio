HEAD SPA SALON WEBSITE

ヘッドスパサロンのウェブサイト（レスポンシブ）
概要
架空の代官山にあるプライベートヘッドスパ専門店のサイト
担当内容
企画、デザイン、コーディング
制作期間
企画、設計、WF：5日
デザインカンプ：4日
コーディング：1週間
使用ツール
Figma / Illustrator / Photoshop / VSCode ( HTML / CSS / JavaScript )
URL
https://intp.site/2522/spasalon

DESCRIPTION

ターゲット
30代〜50代の都内で働く女性
リラクゼーションへの意識が高く、癒しを求めている方々
サイトの目的
・新規顧客の集客（美容ポータルサイトでは伝わりづらいサロンの世界観を発信。店舗のコンセプトや施術の効果を伝えることで、予約への誘導を強化）
・既存顧客の再来率向上（ニュースやブログを活用し、季節の特別メニューや美意識が高まる豆知識などの情報を継続的に発信）
情報設計
サロンのコンセプトを全面に伝えるように、メインビジュアルには森の動画を配置し自然の癒しを演出。お得情報やサロン定休日などの重要な情報が優先的に伝わるように、ニュースをメインビジュアル直下に最新の3件のみ設置するなど、トップページは必要な情報を厳選し予約や詳細ページへスムーズに遷移できる導線を設計。また、施術メニューの詳細やスタッフ情報を掲載し、安心して予約できるよう配慮する。
デザイン
「上質感」と「透明感」のあるデザインを目指し、落ち着いたアースカラーを基調に、視認性を高めるために適切な余白を設け、読みやすいフォントを採用。施術のリラックス感を伝えるために水や静けさを感じさせるビジュアルを用い、シンプルでありながら高級感のあるデザインを実現。サイト全体にスムーズなアニメーションを取り入れ、静かで心地よい印象を与えるUIを設計。
