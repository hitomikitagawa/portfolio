PORTFOLIO SITE

ポートフォリオサイト（レスポンシブ）
概要
自身のポートフォリオサイト
担当内容
企画、デザイン、コーディング
制作期間
企画、設計、WF：1週間
デザインカンプ：1週間
コーディング：10日間
使用ツール
Figma / Illustrator / Photoshop / VSCode ( HTML / CSS / JavaScript )
URL
https://intp.site/2522/portfolio

DESCRIPTION

ターゲット
・企業の採用担当者様、および現場のデザイナー様
・ポートフォリオから応募者のポテンシャルや姿勢などを見極めたいとお考えの方々
サイトの目的
・制作背景や意図を明示し、自身がどのような価値観でデザインに向き合っているかを伝えることで、人柄や考え方に興味を持ってもらうこと。
・制作物やこのポートフォリオ全体を通して、「Webサイトの構成から実装まで一貫して1人で行えること」に加え、「デザインのバランス感覚」や「ユーザー視点に基づいた導線設計」などのスキルを伝えること。
情報設計
テーマは、『直感的に見やすいこと』と『快適な使用感』

採用担当者の方々が限られた時間の中でスムーズかつ快適に必要な情報にアクセスできるよう、情報を絞って簡潔に配置。要素同士の関係性や優先順位が直感的に伝わるよう、視線の動きや配置のリズムにも配慮した。

・TOPページ・・・縦に長く間延びしないよう、作品紹介を分離してWORKSページに集約。
・WORKSページ・・・「WEB DESIGN」と「GRAPHIC DESIGN」の2つカテゴリーに分け、Web作品は詳細ページに遷移、Graphic作品はファンシーボックスによるポップアップ表示にすることで、離脱や戻る手間を減らし、ストレスの少ない閲覧体験を実現。
・ABOUTセクション・・・自己紹介や経歴は簡潔にまとめつつ、PERSONALITYなどの補足情報はタブに分けて整理。必要な情報を選んで読める構造とすることで、読み手にとって負担の少ない閲覧を可能にした。

全体を通して、ページ遷移や動きはシンプルかつ軽量にし、採用担当者がPC・スマホのどちらで見てもストレスなく動作するように意識した。
デザイン
テーマは『コンテンツを引き立てるデザイン性』と『細部へのこだわり』

私は、流行のデザインや凝ったアニメーションが好きで、当初は「個人のポートフォリオなのだから、自分の好みを全面に出してもよいのでは」と考えた。
しかし、目的はあくまで「採用担当者に自分を知ってもらうこと」。ユーザー視点を欠いた表現では本質から外れてしまうと感じ、数多くのポートフォリオやWebサイトを見て構成・導線・演出を分析し直した。
今回のデザインでは、ポートフォリオとしての目的や課題を明確にしたうえで、それを支えるための設計を重視。主張しすぎず内容を引き立て、「読みやすさ」や「わかりやすさ」を支えるビジュアルを目指した。

・レイアウト・・・コンテンツが自然と際立つよう、十分な余裕をもたせた余白をとり、自然と視線を運べる構成にしている。
・フォント・・・品のある柔らかさをもつ欧文フォント「Cormorant」と、読みやすく美しい和文フォント「Noto Serif JP」を組み合わせ、可読性とフォーマルさを両立。
・配色・・・背景に淡い暖色系のグレーに和紙のような質感を加え、デジタル上でも紙の温かさや親しみやすさが感じられるよう工夫。アクセントカラーは全体に調和する彩度を落としたイエローを採用し、「未来」や「希望」といった前向きなニュアンスを込めている。
・アニメーション・・・ボタンのホバー時の反応や視線誘導のための動きなど、装飾ではなくユーザーの理解や操作を補助する役割を担っている。目を引くための演出ではなく、「伝えるための動き」として細部まで配慮した。
