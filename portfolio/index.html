SKILLSできること

企画・コンサルティング経験を活かして
機能的＋価値が伝わるデザインをご提供します

Web Design

クライアントの課題解決や価値を高めるウェブデザインを目指します。また、UI・UXを考え、ユーザーが操作しやすくターゲットに刺さるデザインを創ります。

Coding

HTMLやCSSを用いた正確なデザイン再現に加え、JavaScriptによる動的な演出やレスポンシブ対応も可能です。見た目の美しさだけでなく、管理しやすいコードを意識し構築します。

Graphic Design

ブランドのコンセプトや背景を大切に価値が伝わるデザインを創ります。ロゴやバナーはもちろん、ポスターやメニュー表、POPなど販促物も作成可能です。

ABOUT ME私について

北川 ひと美Kitagawa Hitomi

大学卒業後、新卒で化粧品メーカーに入社し、9年間美容室や代理店へのコンサルティング営業・教育支援に従事。ヘアケアブランドの販促ツール（パンフレットやPOPなど）の企画・制作に携わる中で、ブランド価値を伝えるデザインに関心を持ち、独学で学び始める。
その後、配偶者の海外駐在に伴い、オーストラリア・タイへ移住。個人ECサイト運営やジュエリーメーカーにて営業、進行管理、デザイン調整などに携わる。
2024年からスクールでWebデザインやコーディングを学ぶ。心理学を専攻していたこともあり、UI/UXに関心が高い。これまでの経験を活かしユーザー視点に立った課題解決型のデザインを目指していきたい。
